# Tahcia Spec

Canonical specification for the Tahcia automation ecosystem.

Contains:

- command definitions
- JSON schemas
- examples
- snippets
- AI metadata
- protocol structure

Used by:

- Tahcia CLI
- MCP integrations
- AI agents
- editor tooling
- automation runtimes

## Purpose

The Tahcia Spec repository provides a single source of truth for command behavior and workflow structure across the ecosystem.
